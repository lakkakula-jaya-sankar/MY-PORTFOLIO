Online Book Store

The Online Book Store is a web-based application that allows users to browse, search, and purchase books online. The project is built using Java Servlets, JSP, and MySQL, providing a dynamic and interactive user experience. The system facilitates book management, user authentication, order processing, and payment handling.

Expense Tracker

The Expense Tracker is a web-based application designed to help users manage their daily expenses efficiently. Developed using HTML, CSS, JavaScript, and Bootstrap, this application allows users to log their expenses, categorize them, and track their spending over time. The system provides an intuitive user interface for better financial management.

Weather Web App

Built a responsive weather web application using React and Axios to fetch and display real-time weather data. The app updates dynamically based on user input, showing temperature, humidity, wind speed, and weather conditions. Used React hooks for state management, Axios for API integration, and implemented error handling for invalid inputs and network issues, ensuring a clean and user-friendly UI.
